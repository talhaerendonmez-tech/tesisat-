🤖 Yapay Zeka Bağlantısı

Asistana gerçek AI gücü vermek için ücretsiz bir API anahtarı bağla. Anahtar sadece bu cihazda saklanır.

📌 Anahtar nasıl alınır? (1 dakika)
Gemini için:
  1. aistudio.google.com/apikey adresine git
  2. Google hesabınla gir → Create API Key'e tıkla
  3. Çıkan uzun anahtarı buraya yapıştır
Groq için:
  1. console.groq.com/keys adresine git
  2. Hesap aç (ücretsiz) → Create API Key'e tıkla
  3. Anahtarı buraya yapıştır

İkisi de kredi kartı istemez ve günlük cömert ücretsiz limitleri vardır. Anahtarın gizli kalması için kimseyle paylaşma.
