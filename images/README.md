# Images / Görseller

Bu klasöre aşağıdaki görseller eklenmelidir:

## Zorunlu Dosyalar
- `favicon-16.png` — 16x16 piksel favicon
- `favicon-32.png` — 32x32 piksel favicon
- `apple-touch-icon.png` — 180x180 piksel Apple cihazlar için ikon
- `icon-192.png` — 192x192 piksel PWA ikonu
- `icon-512.png` — 512x512 piksel PWA ikonu
- `og-image.jpg` — 1200x630 piksel sosyal medya paylaşım görseli

## Opsiyonel Dosyalar
- `hero-bg.jpg` — Hero bölümü arka plan görseli (varsa)
- `logo.svg` — Vektörel logo dosyası
- `about-photo.jpg` — Hakkımızda bölümü görseli
